# Rule-Based AI Chatbot 🤖

A beginner-level Artificial Intelligence project developed using Python.  
This chatbot responds to predefined user inputs using simple `if-else` logic and runs continuously until the user exits the program.

---

## 📌 Features

- Handles greetings like Hello and Hi
- Responds to predefined questions
- Uses `if-else` decision-making logic
- Runs in a continuous loop
- Handles exit commands like `bye` and `exit`
- Case-insensitive user input handling
- Removes punctuation for better interaction

## ▶️ How to Run the Project

1. Install Python on your system
2. Open the project folder in Visual Studio Code
3. Open terminal
4. Run the following command:

```bash

```
